Abbreviation/Acronym Key

GS Game Started Games started as the pitcher.
GP Game Pitched The total number of games in which the pitcher appeared, as the starter or reliever.
W Win The number of wins credited to a pitcher.
L Loss The number of losses charged to a pitcher.
SV Save The number of saves credited to a pitcher.
ERA Earned Run Average The average number of earned runs allowed by a pitcher; total number of earned runs allowed multiplied by 6 divided by innings pitched ((ERx6)/IP).
IP Inning Pitched The total number of innings pitched by the pitcher. The pitcher is credited for one-third inning for each out he records.
H Hit The number of hits allowed by a pitcher.
R Run The number of runs allowed by a pitcher.
ER Earned Run The total number of earned runs allowed by the pitcher.
K Strikeout The number of strikeouts by a pitcher.
E Error Total errors commited by player.
